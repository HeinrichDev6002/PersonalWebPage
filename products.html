Web

Landing Page
Landing page é uma página web independente criada para uma campanha específica com o objetivo único de converter visitantes em leads ou clientes. Ela foca em uma única ação, como preencher um formulário para baixar um e-book ou comprar um produto, e é projetada para eliminar distrações, como menus de navegação, e guiar o usuário para essa meta.
E-Commerce
Website dedicado a venda de produtos pela internet, eliminando a necessidade de um ponto de venda físico. Ideal para quem deseja expandir o alcance das vendas alcançando pessoas de diversos
API Rest
Aplicações web seguindo o padrão Rest. Ideal para quem necessita de serviços mais elaborados, que exijam que dados e informações sejam mantidos salvos.
Portfólio
Website dedicado a apresentação profissional e dos trabalhos deste profissional, é indicado para todo tipo de profissional, em especial artistas, programadores, designers e outros pois permite mostrar seu trabalho de forma eficiente.
